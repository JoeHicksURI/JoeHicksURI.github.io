Final Project

This is my final project for CSC 106. It is a periodic table of elements created on the Khan Academy Website using processing.js The program uses object oriented programming. The program displays a periodic table of elements. Each element is a clickable button and shows details about the element when clicked. The program also displays a simplified Bohr model of the atom when you click it. A video showing a walk through of the program is embedded below.
